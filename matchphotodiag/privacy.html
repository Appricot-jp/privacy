プライバシーポリシー

本ポリシーは、本アプリ「マチアプ写真診断 (MatchAppPhotoDiagnose)」(以下「本アプリ」) における個人情報および利用者データの取扱いについて定めるものです。本アプリは利用者のプライバシーを最大限尊重し、最小限のデータ取扱いを基本方針としています。

取得する情報

本アプリは以下の情報を取り扱います。

  • 写真ライブラリへのアクセス: 診断対象の写真選択のために使用します。アクセスは利用者の明示的な許可が必要です。
  • 購入情報: 本気診断パックの購入有無および残クレジット数は、端末内 (UserDefaults) にのみ保存されます。
  • 広告識別子 (IDFA): 利用者が App Tracking Transparency の許諾ダイアログで許可した場合に限り、広告配信パートナー (Google AdMob) によって関連性の高い広告表示と効果測定のために使用されます。許可しなかった場合は使用しません。

本アプリは氏名・メールアドレス・電話番号・住所などの個人を識別する情報を取得しません。アカウント登録も不要です。

写真データの取扱い

利用者が選択した写真は、診断のために前処理 (リサイズ・圧縮) された上で、Google が提供する Gemini API に HTTPS で送信されます。

  • 送信されるのは前処理後の画像データのみで、位置情報・撮影日時等の Exif メタデータは含めません。
  • Gemini API への送信および同 API におけるデータの取扱いについては、Google のプライバシーポリシー (https://policies.google.com/privacy) が適用されます。
  • 運営者は写真データを自社サーバーには保存しません。診断完了後、送信した画像は API レスポンス受領後にメモリ上から破棄されます。

広告について

無料診断では Google AdMob のリワード広告およびバナー広告を表示することがあります。AdMob は広告配信のために広告識別子 (IDFA) や端末情報を利用する場合があります。詳細は Google の広告ポリシー (https://policies.google.com/technologies/ads) をご参照ください。

本気診断パックを購入された利用者には、当該残クレジットを利用する診断において広告を表示しません。

第三者提供

運営者は、利用者から取得した情報を以下の場合を除き、第三者に提供しません。

  • 利用者本人の同意がある場合
  • 法令に基づき開示が必要な場合
  • Gemini API・AdMob 等、本アプリの機能提供に必要な範囲で外部サービスに送信する場合

データの削除

本アプリをアンインストールすることで、端末に保存された購入クレジット情報・診断履歴・各種設定はすべて削除されます。設定画面からも個別にリセットが可能です。

お問い合わせ

本ポリシーに関するお問い合わせは、App Store の本アプリページに記載のサポート連絡先、または サポートページ をご参照ください。

ポリシーの改定

本ポリシーは、法令の改正やサービス内容の変更に応じて改定されることがあります。改定後はアプリ内および本ページに表示された時点から効力を生じます。
