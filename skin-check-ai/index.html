プライバシーポリシー

肌診断AI スキンチェック

最終更新日: 2026年2月19日

本プライバシーポリシーは、「肌診断AI スキンチェック」（以下「本アプリ」）におけるユーザー情報の取り扱いについて定めるものです。本アプリをご利用いただく前に、本ポリシーをよくお読みください。

1. 収集する情報

1-1. 顔写真データ

本アプリは、肌状態の分析を目的として、ユーザーがカメラで撮影した写真またはフォトライブラリから選択した写真（以下「顔写真データ」）を収集します。

重要: 本アプリは顔認識・顔検出技術（Apple Vision Framework等）は使用しておらず、フェイスマップ、深度データ、生体認証データなどは一切収集しません。収集するのは肌分析に使用する写真画像のみです。

1-2. 肌プロフィール情報（任意）

美肌+プラン（有料プラン）をご利用の場合、より正確な診断のために以下の情報を任意でご登録いただけます：

  • 肌タイプ（乾燥肌、脂性肌など）
  • 年代
  • 性別
  • 肌の悩み
  • 現在のスキンケア

1-3. 診断結果データ

AI分析により生成された肌スコア（バリア機能、水分量、油分量、毛穴、ハリ・弾力、透明感、ターンオーバー、炎症）およびアドバイステキストを保存します。

2. 情報の利用目的

収集した情報は、以下の目的にのみ使用します：

  • 肌状態の分析: 顔写真データをAI（OpenAI API）に送信し、肌状態のスコアリングとアドバイスの生成を行います
  • 診断履歴の管理: 過去の診断結果をデバイス上に保存し、肌状態の変化を追跡できるようにします
  • パーソナライズ: 肌プロフィール情報を基に、個人に最適化されたアドバイスを提供します（美肌+プランのみ）
顔写真データを広告ターゲティング、ユーザー識別、顔認識、その他分析以外の目的に使用することは一切ありません。

3. 第三者への提供

3-1. OpenAI（AI分析）

肌分析のため、顔写真データをOpenAI社のAPI（api.openai.com）に送信します。OpenAIはAPIリクエストで送信されたデータをモデルのトレーニングに使用しません（OpenAI API Data Usage Policy に基づく）。送信されたデータはAPI処理完了後、OpenAIのデータ保持ポリシーに従い最大30日間保持された後、削除されます。

3-2. Google AdMob（広告配信）

無料プランでは、Google AdMob SDKを使用して広告を配信します。AdMobは広告配信に必要な情報（デバイス識別子等）を収集する場合がありますが、顔写真データはAdMobと共有されません。

3-3. Apple（アプリ内課金）

美肌+プランの購入処理は、AppleのStoreKitを通じて行われます。課金に関する情報はAppleが管理します。

上記以外の第三者に顔写真データを共有・販売・提供することはありません。

4. データの保存場所

  • 顔写真データ: ユーザーのデバイス内（アプリのDocumentsディレクトリ）にJPEG形式で保存されます。iCloudへの自動同期は行いません。
  • 診断結果: ユーザーのデバイス内にJSON形式で保存されます。
  • 肌プロフィール: ユーザーのデバイス内（UserDefaults）に保存されます。
  • サーバー: 本アプリは独自のサーバーを持たず、ユーザーデータをサーバーに恒久的に保存することはありません。

5. データの保持期間

  • デバイス上の顔写真・診断結果: ユーザーが手動で削除するまで保持されます。アプリ内の履歴機能から、個別または一括で削除できます。
  • OpenAI APIに送信されたデータ: OpenAIのデータ保持ポリシーに従い、APIリクエスト処理後、最大30日間保持された後に削除されます。
  • アプリの削除: アプリを削除すると、デバイス上のすべてのデータ（写真、診断結果、プロフィール）が完全に削除されます。

6. ユーザーの権利

ユーザーは以下の権利を有します：

  • データの閲覧: アプリ内の履歴機能で、すべての診断結果と保存された写真を閲覧できます
  • データの削除: アプリ内から個別または一括でデータを削除できます
  • 写真撮影の拒否: カメラやフォトライブラリへのアクセス許可をいつでも拒否・取り消すことができます（iOS設定から変更可能）
  • プロフィール情報のリセット: 肌プロフィール情報はいつでもリセットできます

7. セキュリティ

本アプリは以下のセキュリティ対策を講じています：

  • API通信はHTTPS（TLS暗号化）により保護されます
  • データはユーザーのデバイス内にのみ保存され、外部サーバーに恒久的に保存されることはありません
  • APIキーはKeychainにより安全に管理されます

8. 未成年者のプライバシー

本アプリは13歳未満のお子様を対象としておらず、13歳未満のお子様から意図的に情報を収集することはありません。13歳未満のお子様が本アプリを使用していることが判明した場合は、お問い合わせください。

9. プライバシーポリシーの変更

本ポリシーは、法令の変更やサービス内容の変更に伴い、事前の通知なく変更される場合があります。重要な変更がある場合は、アプリ内でお知らせします。最新のポリシーは常にこのページでご確認いただけます。
